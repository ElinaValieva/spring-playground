### Prerequsites :heavy_exclamation_mark:

 1. `JDK 1.8` at least - for running app
 2. `Maven 3+`            - for building package


### Notification configuration 
Server `application.properties`: 
- `spring.mail.host` 
- `spring.boot.admin.notify.mail.to`
